Chapters

Interactive chapters

Better than show notes! Always let the audience know where they are by the highlighted chapter.

const instance = new Shikwasa({
  audio: {
    src: '...',
    chapters: [
      {
        title: 'Final Q&A',
        startTime: 2300,
        endTime: 4500,
      },
    ]
  },
})

Jump to the desired chapter in no time with updateChapter.
